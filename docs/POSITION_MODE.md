# 포지션 모드 — One-way vs Hedge (coin toggle)

## 요약

| 구분 | 체결 규칙 | 롱+숏 동시 |
|------|-----------|------------|
| **코인 (기본)** | **원웨이** — 반대 청산 후 잔량 전환 | 불가 |
| **코인 + 헷지 모드 ON** | 레그별 독립 진입·청산 | **가능** |
| **해외선물·주식 등 4상품군** | **원웨이** (코인 기본과 동일) | 불가 |

---

## 원웨이 체결 (전 상품 공통 로직)

1. `청산량 = min(주문수량, 반대보유)`
2. `잔량`만 새 방향 체결

### 예시

- 선물: 매수 20 → 매도 10 ⇒ **매수 10**
- 주식: 100주 매수 → 130주 매도 ⇒ **100주 청산 + 30주 매도**

구현: `fillOneWayLeg` (`useHedgeLegs: false`)

## 헷지 모드 (코인선물만) — 거래소 방식

- UI: **「헷지 모드」** 토글 (기본 OFF, 코인만)
- ON 이후 우측: **매수 / 매도** (매수전환·매도전환·4버튼 없음)
- 체결 규칙 (`fillHedgeExchange`):
  - **매수**: 숏 있으면 숏 청산 → 남은 수량은 롱 진입/추가
  - **매도**: **숏 진입/추가만** (롱은 여기서 청산하지 않음)
  - **롱 청산**: 호가 평단 줄 **「청산」** 버튼 또는 아래 청산주문 ON 시 왼쪽 칸
  - 롱·숏 **동시 보유** 가능
- OFF: **원웨이** + 우측 **시장가/지정가·매수/매도** (호가 클릭=지정가, 방향 자동)

### 호가 — 청산주문 (헷지 ON일 때만, 호가 최상단 바)

| 청산주문 | 호가 왼쪽 (매도 칸) | 호가 오른쪽 (매수 칸) | 평단 줄 「청산」 |
|----------|---------------------|------------------------|------------------|
| **OFF** | 숏 진입 | 숏 청산 → 롱 | 롱 **전량** 청산 |
| **ON** | 롱 청산 (공유 수량) | 숏 청산 (공유 수량) | (동일) |

- 공유 수량: `sharedOrderQty` (우측 수량·속도주문·단축키 `1` `2` `3`)
- 헷지 ON 시 **시장 체결** 테이블은 호가 **바로 아래** (`pro-table`). 왼쪽 tape 패널은 중복 방지로 숨김.

구현: `fillHedgeExchange` — `src/engine/hedgeExchangeFill.ts` · UI — `OrderBookLadder.tsx`

### 호가 STOP · TP/SL 열

| 열 | 클릭 | 마커 표시 |
|----|------|-----------|
| **왼쪽 STOP** | MIT/STOP 등록 (매수전환→LONG, 매도전환→SHORT) | MIT/STP 뱃지 |
| **오른쪽 TP/SL** | **선택 포지션** 있을 때 클릭 → TP/SL 1건 등록 | TP·SL 뱃지 (해당 가격 줄) |

**호가 TP/SL (숏 예시):** 평단 **아래** 클릭 → **TP**, **위** 클릭 → **SL**.  
**OCO:** 익절·손절 중 **하나** 터지면 같은 포지션의 나머지 보호주문 **전부 취소**.

익절/손절 틱 자동: 포지션 선택 → 우측 **자동익절/손절** → **등록** (기존).

### 주문 취소 (MIT/STOP·미체결)

- 호가 STOP 열 마커 **×** 또는 **우클릭** (`WAITING`/`ARMED`만)
- 오른쪽 **MIT / STOP** 패널 **×**
- 하단 **미체결** 탭 — 줄 **취소** / **전체취소** (가장 쉬움)
- 호가 **즉시 체결** 주문은 미체결에 없음 → 취소 대신 포지션 **청산**
